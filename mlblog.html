# Probability

Probability is the bedrock of ML, which tells how likely is the event to occur. The value of Probability always lies between 0 to 1. It is the core concept as well as a primary prerequisite to understanding the ML models and their applications.

Some topics in probabilistic methods for machine learning might include:

  • density estimation,
  • kernel density estimation,
  • divergence estimation, etc.

This would also include techniques such as Naive Bayes and graphical models such as Bayesian belief networks.

Sample Space => Sample space is all the possible outcomes of an event .

For example, if you roll a dice, 6 things could happen. You could roll a 1, 2, 3, 4, 5, or 6. Sample space is 6

A continuous random variable is a random variable that has a real numerical value.

  • PDF: Probability Density Function, returns the probability of a given continuous outcome.
  • CDF: Cumulative Distribution Function, returns the probability of a value less than or equal to a given outcome.
  • PPF: Percent-Point Function, returns a discrete value that is less than or equal to the given probability.

Cumulative Distribution Function(CDF)

The cumulative distribution function is used to describe the probability distribution of random variables. It can be used to describe the probability for a discrete, continuous or mixed variable. It is obtained by summing up the probability density function and getting the cumulative probability for a random variable.

A probability measure is defined on events.But once we map it to the real line By the CDF .

FX(x)=P(X≤x), for all x∈R. Note that the subscript X indicates that this is the CDF of the random variable X.

The Probability Density Function(PDF) defines the probability function representing the density of a continuous random variable lying between a specific range of values.

Cost Function/Loss function:

The terms cost function & loss function are analogous. Loss function: Used when we refer to the error for a single training example. Cost function: Used to refer to an average of the loss functions over an entire training dataset.

Understanding Cost Functions

Gradient Descent:

Gradient descent is an iterative optimization algorithm for finding the local minimum of a function. To find the local minimum of a function using gradient descent, we must take steps proportional to the negative of the gradient (move away from the gradient) of the function at the current point.

Gradient descent (GD) is an iterative first-order optimisation algorithm used to find a local minimum/maximum of a given function.
