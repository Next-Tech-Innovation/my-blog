How it's Done

"For blur-box, I'm using ::after to style and control the image background to make it blur just only inside the box for the content. For all you have to do is just set width and height of the element that you want to and use class ".box-blur". for profile images, You should use images with a specific size ( 360px width and 240px height) so the images won't be cropped out. the blur background panel will use the images that you use for blury panel background (you dont have to do anything. Just set src of the image that you are going to use.)"
